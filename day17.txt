모듈(module)
늘 만들던 파일 안에 프로그램.
-독립된 하나의 소프트웨어/하드웨어의 단위
-파이썬의 파일(.py)은 모듈이 될 수 있다.
-import 키워드를 사용하여 같은 패키지 안에 있는 다른 모듈(파일)을 지정해서 읽어들이는 처리를 한다.

모듈의 사용
1. import (파일이름)
-파일 안에 있는 모든 것을 사용한다.
2.from (파일명) import *
-from 뒤에 있는 파일에서 모든 것(*)을 사용한다.
3.from (파일명) import (함수이름...)
-특정 함수만 골라서 가져오는 방법으로 import 뒤에 함수 이름을 한 개 또는 여러 개를 적어서 사용한다.
