문자열 자료형의 연산
더하기: `문자열 + 문자열` 형태로 사용하며, 문자열을 하나로 합친다.
곱하기: `문자열 * 정수` 형태로 사용하며, 문자열을 정수만큼 반복시켜서 하나의 문자열로 만든다.
문자열은 포맷팅이 가능하다. (문자열 자료형의 고유 특징!!!)
포맷팅: 문자열 중 변경되지 않는 값을 고정하고 변경되는 값의 위치를 지정하여 출력하는 것
포맷코드를 활용한 포맷팅-구식, %s: 문자열 포맷코드, %d: 정수 포맷코드, %f: 소수 포맷코드
포맷코드: 문자열 안에서 바뀌는 값을 코드를 활용하여 자리를 잡는 행위
f-string()함수를 활용한 포맷팅-신식, 가장 많이 사용되고 있음.

입력함수
입력: 사용자가 값을 직접 키보드를 통해 입력하여 값을 입력하는 행위
input()함수를 활용한다.
값을 변수에 저장하는 방법은 2가지가 있다.
1. py파일에서 직접 값을 입력하여 저장하는 방법-실행 전
2. input()함수를 활용하여, 실행하고 값을 입력하여 저장하는 방법-실행 후
