리스트 자료형의 함수
-사용법: `리스트 변수 이름.함수()`
-append(값): 값을 리스트의 맨 뒤에 추가한다.
-insert(인덱스, 값): 값을 해당 인덱스에 밀어내고 추가한다.
-remove(값): 값을 리스트에서 하나 삭제한다. (단, 해당 값이 없다면 오류 발생)
-pop(인덱스): 해당 인덱스의 요소를 삭제한다. 만약 인덱스의 값을 넣지 않으면 맨 마지막 요소를 삭제한다. 값을 삭제하고 반환한다.
-sort(): 리스트 값들을 오름차순으로 정렬한다.
-sort(reverse=True): 리스트 값들을 내림차순으로 정렬한다.
-clear(): 리스트의 값들을 전체 삭제한다.
-값 in 리스트: 해당 값이 리스트 안에 있는지 판별하는 코드
-값 not in 리스트: 해당 값이 리스트 안에 없는지 판별하는 코드
-count(값): 리스트 안에 해당 값이 몇 개 있는지 파악하여 수를 반환하는 코드
-reverse(): 리스트의 값들을 역순으로 만든다.
-연산이 가능

튜플 자료형
-값을 중심으로 양 옆에 소괄호 기호가 온다.
-모든 자료형이 표현 가능하며, 쉼표를 통해 구분된다.
-중복 값을 허용한다.
-값을 한 번 생성했으면 수정, 삭제 불가.
-인덱스가 존재.
-연산이 가능.
-절대불변의 값들을 사용하기 위해 존재.
-값이 하나만 올거면 끝에 쉼표를 붙여줘야 한다.

튜플 자료형의 함수
-`튜플 변수 이름.함수()` 형태로 사용.
-count(값): 해당 값의 개수를 반환한다.
-index(값): 해당 값의 인덱스 값을 확인.

집합 자료형
-중복을 허용하지 않는다.
-순서가 없다. > 값을 활용할 수 없다.
-값을 중심으로 양 옆에 중괄호 기호로 표현.
-집합 자료형은 set(값) 함수를 활용하여 만들 수 있다.
-리스트 > 집합 > 리스트로 중복을 삭제하고 활용 가능. list(): 리스트 자료형으로 변환. list(set())로 쉽게 가능. 단, 순서가 없기에 값의 순서는 랜덤.
