함수: 특정 기능을 실행하는 코드
함수의 종류: 내장함수, 외장함수, 사용자 정의 함수

내장함수:
-파이썬에서 항상 사용할 수 있는 함수가 내장 되어있다.
-import할 필요가 없다.(외부 모듈에서 저장된 함수를 불러와서 사용할 때 필요한 명령어)
*모듈: .py파일 자체

문자열 내장함수:

-chr(): 모든 문자는 자신만의 코드값(아스키코드, 유니코드를 가지고 있다.
특정 문자의 코드 값을 전달하며 해당 유니코드 값을 가진다.
문자를 반환하는 함수

-eval(): 실행하고자 하는 표현싪들 문자열로 전달하면 표현식의 결과값을 반환하는 함수

-str(): 소괄호 안에 전달받은 인자를 문자열로 반환하는 함수
문자열로 형변환하는 함수


슷자형 내장함수:

-abs(): 소괄호 안에 전돨된 인자의 절댓값을 반환한다.

-divmod(): 소괄호 안에 전달된 두 인자를 나누어서 몫과 나머지를 하나의 튜플 결과로 반환한다.

-float(): 소괄호 안에 전달된 인자를 실수로 만들어서 반환한다.
소수로 형변환
전달된 인자가 없을때는 0.0반환

-int(): 소괄호 안에 전달된 인자를 정수로 만들어서 반환한다.
정수로 형변환
전달된 인자가 없을때는 0 반환

-pow(): 소괄호 안에 전달된 두 인자의 거듭제곱을 반환한다.
산술연산자 **기호와 동일

-round(): 소괄호 안에 전달된 인자를 이용해서 반올림한 값을 반환한다.
전달된 인자가 1개이면 정수로 반올림한 값을 반환한다.
전달된 인자가 2개이면 두번째로 전달된 인자만큼 소수점을 남긴다.

인자 vs 파라미터
인자, 인수: 함수 실행시 매개변수에 입력되는 값을 의미한다.
파라미터, 매개변수: 함수 선언시 함수 이름 옆 소괄호 안에서 정의되는 변수들을 의미
함수 선언시 수행문에서 사용될 변수들을 의미

시퀀스 내장함수:
-enumerate(): 리스트와 함께 사용할 수 있다.
저장된 요소와 해당 요소의 인데스를 튜플 형태로 출력

-range(): 3개의 요소로 값의 범위를 나타내고 하나의 리스트로 정리된다.
특정 범위의 값을 생성하기 때문에 범위 생성자
다른 자료형을 형변환이 가능

-len(): 소괄호 안에 전달된 객체의 길이(=값의 개수)를 반환하는 함수

-sorted(): 소괄호 안에 전달된 객체를 오름차순으로 정리한 결과로 반환한다.
소괄호 안에 인자를 하나 받고, 그 뒤에 reverse=True라는 값을 추가하면 내림차순으로 정리한 값을 반환한다.
*sort(): 결과값이 없이 실행하고 끝.
*sorted(): 결과값이 있음.

-zip(): 소괄호 안에 전달된 여러 개의 객체의 각 요소를 튜플로 묶어서 반환하는 함수.
전달된 객체의 길이가 서로 다르다면 길이가 짧은 객체를 기준으로 동작한다.

함수 vs 매서드
함수: 어떠한 기능을 하는 명령어.
클래스 영역 바깥에서 선언됨.
매서드: 함수와 동일.
클래스 영역 안에서 선언됨.

문자열 매서드
-.format(): 문자열을 정렬하는 방법에 대한 매서드
옵션  의미
>    지정된 공간 내에서 왼쪽 정렬
<    지정된 공간 내에서 오른쪽 정렬
^    지정된 공간 내에서 가운데 정렬
