1. 문자열 메서드
(1) .count(문자열)
: 문자열에서 소괄호 안에 오는 문자열의 개수를 반환하는 메서드
: 반환 값은 정수이다.

(2) .find(문자열)
: 문자열에서 소괄호 안에 오는 문자열의 위치(Index)를 반환하는 메서드
: 문자열 값이 존재하지 않으면 -1 반환
: 반환 값은 정수이다.

(3) .upper()
: 문자열 값을 대문자 형태로 반환

(4) .lower()
: 문자열 값을 소문자 형태로 반환

(5) .isupper()
: 문자열 값이 대문자인지 판단하여 True/False결과로 반환

(6) .islower()
: 문자열 값이 소문자인지 판단하여 True/False결과로 반환

(7) 문자열값.join(문자열변수)
: 점 앞에 있는 문자열 값을 소괄호 안에 있는 문자열 또는 문자열 변수의 각각의 요소 사이에 문자열 값을 포함시킨다.

(8) .split(문자열)
: 하나의 문자열을 소괄호 안에 오는 문자열 값을 기준으로 여러개의 문자열로 분리하여 리스트로 반환한다.

(9) .replace(목표문자열, 대체문자열)
: 문자열 안에서 목표 문자열의 값을 대체 문자열로 변환시켜 문자열로 반환한다.

(10) .strip(문자열)
: 문자열 !!양쪽 끝!!에서 소괄호 안에 있는 문자열을 제거한 결과를 반환한다.

(11) .lstrip(문자열)
: 문자열 왼쪽 끝에서 소괄호 안에 있는 문자열을 제거한 결과를 반환하다.

(12) .rstrip(문자열)
: 문자열 오른쪽 끝에서 소괄호 안에 있는 문자열을 제거한 결과를 반환한다.

(13) .isdigit()
: 문자열이 숫자로만 이루어져 있는지 확인하는 메서드
: 숫자를 제외한 다른 값이 존재하면 False반환, 그렇지 않고 값들이 모두 숫자라면 True반환


숫자열: 숫자, 사칙연산
문자열: '' "", 포매팅, 연산, 인덱스
리스트: [], 값을 수정, 연산, 인덱스
튜플: (), 값을 수정불가, 연산, 인덱스
딕셔너리: {}, key:value한쌍
집합: {}, 중복불가&순서없음
불: true/False


2. 클래스
- 여러 변수와 함수(메서드)를 묶어서 처리할 수 있는 것을 일컫는다.
- 공통된 요소들이 모여있는 하나의 틀

3. 클래스와 객체
- 클래스(class)
: 붕어빵 틀, 똑같은 무엇인가를 계속 만들어 낼 수 있는 설계도
- 객체
: 붕어빵, class에 의해서 만들어진 결과물
- 객체를 생성하기 위해선 클래스가 반드시 선언되어야 한다.
- 클래스를 통해서 객체를 만들고 활용할 수 있는 프로그래밍 언어를 객체 지향 프로그래밍 언어라고 한다.

4. 클래스 선언
class 클래스이름:
    # 수행문
    # 데이터 속성 : 클래스 안에서 사용될 변수
    self.변수

    # 메서드 : 클래스 안에서 선언되는 함수들
    def 메서드이름(self, 매개변수):
        메서드 수행문
- self : 객체 구별용 매개변수(클래스 자신을 나타낸다.)
- self.변수
: 변수를 필드/멤버라고 한다.
: self.변수를 활용하여 접근하니 여러 객체도 한번의 선언으로 구별하여 사용할 수 있다.
- 클래스이름 : 첫글자를 대문자로 한다.

5. 클래스 사용
- 추상적인 개념의 클래스로 객체를 만드는 작업을 객체화라고 한다.
- 객체화 하는 방법
: 객체이름(= 변수이름) = 클래스이름()
- 클래스의 필드(멤버/수행문 위치)에 접근 방법
: 객체이름.변수이름
: 객체이름.메서드이름()

6. 클래스를 사용하는 이유
- 하나의 클래스를 통해 여러개의 객체를 생성하므로 코드의 반복을 줄일 수 있기 때문에 사용한다.

7. 생성자(Constructor)
- 객체화시 처음에 반드시 실행되는 메서드
- 클래스를 통해서 객체를 생성하는 순간에 실행되는 메서드이다.
- 클래스 내부에서 생성자를 작성하면 객체를 생성할 때 실행할 코드 작성이 가능하다.
- 생성자는 주로 데이터 속성을 선언 및 초기화시에 사용된다.
- 생성자는 메서드처럼 선언되며 메서드의 이름은 항상 __init__형태로 선언한다.
