상속(Inheritance)
-기존에 사용중인 클래스를 바탕으로 새로운 요소를 추가해서 클래스를 다시 정의하는 것, 클래스를 확장하는 것을 상속이라고 한다.
-새로운 클래스는 기존 클래스의 테이터 속성, 메서드를 상속받는 구조로 되어있다.
-기존 클래스에 새롭게 필요한 데이터 속성, 메서드를 추가해서 코드를 작성 가능하다.
-부모 클래스(슈퍼 클래스): 상속을 하는 클래스
-자식 클래스(서브 클래스): 상속을 받는 클래스
-하나의 부모 클래스는 여러 개의 자식 클래스를 가질 수 있다.
-자식클래스는 하나의 클래스만 상속 받을 수 있다.(단일 상속)
-같은 부모 클래스를 둔 자식 클래스끼리 형제 클래스라 명칭하며 형제 클래스끼리는 상속할 수 없다.

클래스 상속 방법
class 부모클래스:
    수행문

class 자식클래스(부모클래스이름):
    수행문


객체 지향 언어
1. 클래스
2. 상속 ------오버라이딩
3. 다형성 ----오버라이딩
4. 캡슐화

오버라이딩
-부모 클래스에서 정의한 메서드를 자식 클래스에서 재정의하는 것을 말한다.
-같은 이름의 메서드지만 자식 클래스에서는 다르게 동작하게 만든다.
-부모의 메서드를 완전히 새로 덮어쓰거나, super()를 활용하여 부모의 메서드를 활용할 수 있다.
