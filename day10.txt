함수: 어떤 기능을 하는 명령어이며, 객체에 속해있지 않고 단독으로 사용 가능.
입력값과 결과값이 존재.
함수의 입력값은 함수 선언 시 함수 이름 옆에 있는 소괄호 안에 변수가 있는지 없는지에 따라 결정.
함수의 결과값의 유무는 함수의 수행문 영역 안에 return문이 있는지 없는지에 따라 판별.

메서드: 함수와 동일하며, 객체(클래스)에 속해 있어야 하는 함수.
메서드 이름 앞에 .이라는 피리오드를 붙인다.

함수 선언 방법
def 함수이름(변수, 변수...):
    수행문

파라미터(parameter, 매개변수): 함수 선언 시 소괄호 안에 정의되는 변수들
인자, 인수: 함수 사용 시 소괄호 안에 넣어주는 값.
def numberSum(a,b,c): >> 괄호 안 a,b,c는 파라미터
    return a+b+c >> a b c는 인수
