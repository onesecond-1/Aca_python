1. 파일 입출력
-프로그래밍에서 외부 저장장치(하드 디스크)에 데이터를 저장하거나 저장된 데이터를 읽어오는 행위
-내장 함수 open()으로 파일을 열고, 다양한 모드로 읽거나 쓸 수 있다.
-파일 작업이 완료되면 close()로 파일을 닫아주는 것이 중요(자원해제 및 오류 방지)

2. 파일 열기
-open() 내장 함수 사용
-변수 = open("파일이름", "파일열기모드", encoding="utf-8")
# 폴더가 없으면 먼저 만들어야 함. 귀찮으면 파이썬 내 기능프로그램을 설치해야함.
# 파일이름: 파일 경로와 이름을 지정
    ex) C:/doit/새파일.txt >> C드라이브에 doit폴더를 만들고 그 안에 새파일.txt을 만든다.
# 파일열기모드: 읽기 모드, 쓰기 모드, 추가 모드 등
# 읽기모드는 r로 표기, 파일이 먼저 존재해야함.
# 쓰기모드는 w로 표기, 파일이 없으면 새로 생성, 존재하면 기존 내용을 삭제하고 다시 작성
# 추가모드는 a로 표기, 기존 내용을 그대로 유지하고 뒤에 새로운 내용을 추가
# encoding 문자 인코딩으로 한글은 utf-8로 인코딩

3. 파일 쓰기
-쓰기 모드는 파일에 내용을 새로 쓰는 모드
-기존 내용은 지워진다.
-파일열기모드에 w로 표기
-write()함수가 존재, 파일에 내용을 추가
-write()함수는 문자열만 입력 가능, 숫자 표현 시 문자열로 형변환하여 사용

4. 파일 읽기: 읽어서 해당 변수로 이동됨.
-readline(): 한 줄씩 읽기, 줄 바꿈 문자(\n)을 인식하여 줄을 내리며 출력
-readlines(): 모든 줄을 한번에 읽기, 모든 줄을 리스트로 반환, 리스트의 각 요소는 한 줄의 문자열
-read(): 전체 읽기, 파일 전체 내용을 하나의 문자열로 반환.
-반복문과 readline()으로 모든 줄 읽기.

5. 파일에 내용 추가
-파일에 내용 추가 시에는 파일 열기 모드에 a로 표기
-이미 존재하는 파일의 끝에 내용을 추가하고 싶을 때 사용
-기존 내용은 유지되고, 새로운 내용이 다음 줄부터 덧붙여짐.

6. with 문법으로 파일 자동 닫기
- with문법은 파일을 열고 닫는 작업을 자동으로 처리
-close() 함수를 사용하지 않아도 됨.
-with문법 수행문을 벗어나면 파일이 자동으로 닫힘.
-with open("파일경로 및 이름", "파일 열기 모드", encoding="utf-8") as (변수):
    수행문

7. 파일 처리 시 유의사항
-파일을 열 때는 항상 적합한 모드를 선택해야함.(write모드 쓰면 다 지워지고 새로 써지니깐)
-파일에 쓸 때는 write()함수를 사용하며, 문자열만 쓸 수 있다.
-파일은 작업이 끝나면 항상 닫아야 하며, 가능하면 with문 사용을 권장
-한글 등 문자 깨짐 방지를 위해 encoding='utf-8'설정을 파일 열기 시 지정해야 한다.
